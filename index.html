Base De Datos

Es un cojunto de datos perteneciente a un mismo contexto y almacenados sistematicamente para su posterior uso.

son lo que se utilizan para o gestores de base Datos y que varios usuarios hagan varias operacionessobre ellas al mismo situados en lugares difrentes . se puedes acceder a ellas mediantes treminales o equipos co unprograma llamado clientes.

Gestor de base datos

son un conjunto de progrmas que manejan todo acceso a la base de datos , con el objetivo de servir de interfaz entre esta, el usuario y la aplicacions utilizadas .

Esquema de base datos

representa la configuracion logica de todo o parte de una base de dato relacional . puede existir de dos forma: coo representacion viual y como un cojunto de formuls conocidos como restriciones de integridad que controlan una base d datos . estas formulas se expresan en un lenguaje de definicion , tal como SQL (structure query languje).
